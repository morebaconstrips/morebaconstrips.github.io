Cookie Cleaner

Everyone knows the best part of browsing the internet is the delicious cookies you get from every website you visit. Therefore we have created a special website just for you to get a lot of new cookies. Can you collect all of them before they turn stale? Start the website by clicking the ‘Start Docker container’ button below!

Seems like we need to be faster in order not to let the cookie go bad

import requests

url = 'http://ths.eemcs.utwente.nl:33109/'

cookies = {'password': 'I_LOVE_COOKIES<3'}
response = requests.get(url+'/cookie', cookies=cookies)

password = str(response.content)[1044:1069]

while (True):
    cookies = {'password': password}
    response = requests.get(url+'/cookie', cookies=cookies)
    password = str(response.content)[1044:1069]
    if password[:4] == 'THS{':
        break

print('Flag:', password)
