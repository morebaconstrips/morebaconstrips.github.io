Nyte Encoding

The byte size of 8 bits was chosen because it is a power of 2, other than that, it is an arbitrary length. Instead, we think that the 9-bit byte, i.e. the nyte is much cooler! The string below is a hexadecimal encoding of our flag represented as ASCII in nytes, can you retrieve the flag?

2a120a67b371e4e8653997cc2723297cc4332a15066722f9d0d0403717cc4793a194e67d

flag = ''

hexadecimal = '2a120a67b371e4e8653997cc2723297cc4332a15066722f9d0d0403717cc4793a194e67d'
print('hex:', hexadecimal)

# Transform it into decimal
decimal = int(hexadecimal, 16)
print('decimal:', decimal)

# Transform it into binary
binary = bin(decimal)
print('binary:', binary)

# Split it into chunks of 9 (instead of 8)
sequence = 0
for i in range(len(binary)//9):
    flag += chr(int(binary[sequence:sequence+9], 2))
    sequence += 9
print(flag)
